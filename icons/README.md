# Web Icons

This directory should contain the following icon files for the web app:

- `Icon-192.png` - 192x192 app icon
- `Icon-512.png` - 512x512 app icon
- `Icon-maskable-192.png` - 192x192 maskable icon
- `Icon-maskable-512.png` - 512x512 maskable icon

These icons will be generated automatically when you run `flutter build web`.

For now, Flutter will use default icons if these are not present.
